# MXSpiderCarAndGirl
## 要求
* python3.5+
* requests
* pyquery
* scrapy

## 使用
* 普通方式

```
git clone git@github.com:iamhmx/MXSpiderCarAndGirl.git
cd MXSpiderCarAndGirl
python spider.py
```
* scrapy方式

```
git clone git@github.com:iamhmx/MXSpiderCarAndGirl.git
cd MXSpiderCarAndGirl/scrapy/CarAndGirl/
scrapy crawl carandgirl
```
